BWF Player and Match API

An API that returns data scraped from tournamentsoftware.com formatted in JSON.

Disclaimer: This is a personal project, and is not affiliated with BWF and/or TournamentSoftware in any way, shape, or form.

Data Collection

All data was scraped from various Men's and Women's Singles matches throughout TournamentSoftware as well as the respective players' profiles from from 2008 - 2021. Due to the design of the web-scraper as well as to eliminate any outliers or strange data, some matches have not been added into the database.

Data was populated with an initial scrape up to the time of deployment, and is automatically updated every week with newer matches up to the current day. All data is stored in a private MySQL database.

API Endpoints

Player Search by Name

Search for players by name returns a JSON response with either a list of players that match the search criteria, or an error if no one is found. If no search text is provided, then will return random players.

Query Parameter Description
search_text The name (or part of name) of the player you'd like to search. Defaults to an empty string. (optional)
limit How many results to limit your search by. Defaults to 20. (optional)
Example Query

  • https://baddy-api.com/search?name=lee&limit=2


Example Response

Player Get by ID

Gets a player by their ID (obtainable through the search-by-name API) returns a JSON response with the corresponding player, or an error if no one is found.

Query Parameter Description
player_id The id of the player you'd like to search. (Required)
Example Query

  • https://baddy-api.com/player?player_id=0800FC8B-CBAB-4AC1-B6C7-F3D419906440


Example Response 
  • 
                                {"request":"GET request for single player '0800FC8B-CBAB-4AC1-B6C7-F3D419906440'","status":"success","status_code":200,"data":{"id":"0800FC8B-CBAB-4AC1-B6C7-F3D419906440","country":"KOREA","playHand":"L","birthDate":"1980-04-17","name":"HYUN IL LEE","event":"MS","height":176}}

Matches by Player (and Opponent) IDs

Searches matches containing a player id and an opponent_id (optional) and returns a JSON response of the matches involving the player (and opponent if included), or an error if no one is found.

Query Parameter Description
player_id The id of the player whose matches you'd like to search for. (Required) .
opponent_id The id of the opponent you'd like to search. (optional)
start_year The start year of the period you'd like to search. Defaults to 2008. (optional)
end_year The end year of the period you'd like to search. Defaults to next year. (optional)
limit How many results to return. Defaults to 50. (optional)
Example Query

  • http://localhost:8000/matches?player_id=00127A4D-2889-44A8-921B-7AFFBBC1DE1D&opponent_id=4936E803-3FC7-4B24-9CD8-54BE4B6C2B9E


Example Response 
  • 
                                {"request":"GET request matches between '00127A4D-2889-44A8-921B-7AFFBBC1DE1D' and '4936E803-3FC7-4B24-9CD8-54BE4B6C2B9E'; start of '2008'; end of '2022'; limit of '50'","status":"success","status_code":200,"data":[{"loserId":"4936E803-3FC7-4B24-9CD8-54BE4B6C2B9E","tournamentId":"8385F683-15A3-4E8B-8FBF-EF71992BF553","duration":29,"winnerId":"00127A4D-2889-44A8-921B-7AFFBBC1DE1D"}]}
