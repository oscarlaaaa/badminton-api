BWF Player and Match API

An API that returns data scraped from tournamentsoftware.com formatted in JSON.
Built with: Python, FastAPI, BeautifulSoup4, AWS RDS (MySQL), and Heroku.

Disclaimer: This is a personal project, and is not affiliated with BWF and/or TournamentSoftware in any way, shape, or form. I do not profit from the data scraped.

Data Collection

All data was scraped from various Men's and Women's Singles matches throughout TournamentSoftware as well as the respective players' profiles from from 2008 - 2022. Due to the design of the web-scraper as well as to eliminate any outliers or strange data, some matches have not been added into the database.

Data was populated with an initial scrape up to the time of deployment, and is automatically updated every week with newer matches up to the current day. All data is stored in a private MySQL database hosted on AWS. To view the FastAPI documentation, click here.

API Endpoints

Get Player by ID

Gets a player by their ID (obtainable through the search-by-name API) returns a JSON response with the corresponding player, or an error if no one is found.

Query Parameter Description
player_id The id of the player you'd like to search. (Required)
Example Query

  • http://api.badminton-api.com/player?player_id=0800FC8B-CBAB-4AC1-B6C7-F3D419906440


Example Response 
  • 
                                {"request":"GET request for single player '0800FC8B-CBAB-4AC1-B6C7-F3D419906440'","status":"success","status_code":200,"data":{"id":"0800FC8B-CBAB-4AC1-B6C7-F3D419906440","country":"KOREA","playHand":"L","birthDate":"1980-04-17","name":"HYUN IL LEE","event":"MS","height":176}}

Get Top Winning Players

Search for players by event and returns a JSON response with a list of players with the highest win rates from that event.

Query Parameter Description
event The event you'd like to query. Accepts "MS" or "WS" for Men's Singles and Women's Singles respectively. (Required)
limit How many results to limit your search by. Defaults to 10. (optional)

Get Player Head-To-Head Record

Search for players by ID and returns a JSON response with a list of players along with the wins/losses against those players. Can be sorted in ascending or descending order by wins or losses.

Query Parameter Description
player_id The id of the player you'd like to query. (Required)
wins What parameter to return results by. Accepts 'True' for wins, and 'False' for losses. Defaults to True. (optional)
desc How to order the result parameters by. Accepts 'True' for descending, and 'False' for ascending order. Defaults to True. (optional)
limit How many results to limit your search by. Defaults to 10. (optional)
Example Query

  • http://api.badminton-api.com/player/records?player_id=0800FC8B-CBAB-4AC1-B6C7-F3D419906440&wins=true&desc=true&limit=3

Get Matches by Player ID

Query Parameter Description

start_year The start year of the period you'd like to search. Defaults to 2008. (optional)
end_year The end year of the period you'd like to search. Must be equal or greater than start year. Defaults to next year from present day. (optional)
limit How many results to return. Defaults to 50. (optional)

Tournament Search by Name

Searches and returns a list of tournaments with the given name, within the start and end years, or an error if none are found. If no search text is provided, will simply return a random list of tournaments.

Query Parameter Description
name The id of the tournament you'd like to query. (optional)
start_year The start year of the period you'd like to search. Defaults to 2008. (optional)
end_year The end year of the period you'd like to search. Must be equal or greater than start year. Defaults to next year from present day. (optional)
limit How many results to return. Defaults to 50. (optional)
Example Query

  • http://api.badminton-api.com/tournament/search?name=england&start_year=2020&end_year=2023&limit=50
